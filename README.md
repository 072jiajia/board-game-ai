# Board Game AI

A fun little project where AI plays board games in your browser.

👉 Try it here: https://072jiajia.github.io/board-game-ai/

---

## What is this?

This is a lightweight on-device AI that can play board games like Connect Five.

No server, no backend — everything runs directly in your browser.

It’s basically:
- A tiny AlphaZero-like model
- Running with ONNX Runtime
- Making moves in real time

---

## Why did I make this?

A fun project I originally worked on in college (I initially wanted to make it play Go, but it turned out to be a bit too heavy for my resources 😄), now revisited with newer ideas and wrapped into a simple demo.
